# TaskMaster MCP Bootstrap Assets

This directory stores reusable templates that the skill can deliver to operators after provisioning a TaskMaster MCP server.

- Version: 2.2.1
- Last updated: 04 Nov 2025
- Maintainer: BMAD Skills Core Team

Templates in this folder focus on:
- Capturing environment prerequisites and secrets placeholders
- Documenting provisioning commands and verification steps
- Summarizing hook alignment so BMAD can consume TaskMaster safely

Add or update templates whenever the bootstrap workflow changes.
